adjust peak rate for headcut advance -also adjusts CEAP gully from edge-of-field to trib (assuming rectangular shape and constant tc) new q*qp (m3 * m3/s) equation for entire runoff event compute sediment yield from headcut- assume bd = 1.2 t/m3 assume channel dimensions are same as data file break hydrograph into maxint segments and compute deg at each flow increment calc critical shear and shear on bottom of channel Pa = N/m^2 * m * m/m

degradation of the bank (widening) assume bank shear is 75% of bottom shear calc mass of sediment eroded -> t = cm * m/100cm * width (m) * length (km) * 1000 m/km * bd (t/m3) apply to only one side (perim_bank / 2.) no downcutting below equilibrium slope if bottom shear > d50 -> downcut - widen to maintain width depth ratio if downcutting - check width depth ratio to see if widens

apply to both bank sides

calc mass of sediment eroded -> t = cm * m/100cm * width (m) * length (km) * 1000 m/km * bd (t/m3) adjust for incoming bedload and compute deposition assume bedload is deposited stream is completely filled in compute flood plain deposition output channel morphology compute sediment leaving the channel - washload only - bottom deg is bedload
