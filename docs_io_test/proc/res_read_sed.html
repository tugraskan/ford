subroutine res_read_sed()

Uses

    • input_file_module
    • reservoir_data_module
    • maximum_data_module

~ ~ ~ PURPOSE ~ ~ ~ this subroutine reads data from the lake water quality input file (.lwq). This file contains data related to initial pesticide and nutrient levels in the lake/reservoir and transformation processes occurring within the lake/reservoir. Data in the lake water quality input file is assumed to apply to all reservoirs in the watershed.
